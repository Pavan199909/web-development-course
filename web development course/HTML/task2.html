How to design a favicon using favicon.cc

Following are the steps to follow create favicon

  • step-1: open the browser and search for the favicon.cc
  • step-2: after searching the favicon.cc opening a favicon webpage
  • step-3: In that page we use pencil to draw a particual logo or letters
  • step-4: we have options to pick eraser to clear the mistakes.
  • step-5: Also we have animated creation option to create the icon in animation also.
  • step-6: After creation we can download and paste in head of the html code.
